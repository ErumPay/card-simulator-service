# Card Simulator Seed Data

`SimulatorDataSeeder`가 부팅 시 적재하는 시드 데이터입니다.

- **출처(카드)**: `erumpay-infra/mysql/init/11-seed-card-gorilla-cards.sql` (Card Gorilla 실제 카드)
- **사용자 수**: 10명 (이름은 한글 자음 가-차 슬라이딩: 가나다 → 차카타)
- **카드 수**: 사용자당 3장, 총 30장
- **공통 카드 속성**: `expiry_date=2912`, `cvc=123`, `password_2digit=12`, `card_status=ACTIVE`

| BIN 앞 2자리 | 카드사       | `CardCompany` enum |

| `80`         | 삼성카드     | `SAMSUNG`          |
| `81`         | 신한카드     | `SHINHAN`          |

| `83`         | KB국민카드   | `KB`               |
| `84`         | 롯데카드     | `LOTTE`            |
| `85`         | 우리카드     | `WOORI`            |
| `86`         | 하나카드     | `HANA`             |
